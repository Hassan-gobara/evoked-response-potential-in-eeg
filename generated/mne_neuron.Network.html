    class mne_neuron.Network(params, n_jobs=1)¶

    The Network class.

    Parameters
    paramsdict

    The parameters

    n_jobsint

    The number of jobs to run in parallel

    Attributes
    cellslist of Cell objects.

    The list of cells

    gid_dictdict

    Dictionary with keys ‘evprox1’, ‘evdist1’ etc. containing the range of Cell IDs of different cell types.

    ext_listdictionary of list of ExtFeed.
    Keys are:

    ‘evprox1’, ‘evprox2’, etc. ‘evdist1’, etc. ‘extgauss’, ‘extpois’

    spiketimestuple (n_trials, ) of list of float

    Each element of the tuple is a trial. The list contains the time stamps of spikes.

    spikegidstuple (n_trials, ) of list of float

    Each element of the tuple is a trial. The list contains the cell IDs of neurons that spiked.
